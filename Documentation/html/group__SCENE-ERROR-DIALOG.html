Detailed Description

This scene is called when the user tries to enter an application mode (i.e SCENE-USB-CAN) from SCENE-START but USB link is busy.

void usb_can_scene_error_dialog_on_enter ( void *  context)

This function is called when SCENE-ERROR-DIALOG is entered. It will display 2 popup messages and an icon.

This function:

  1. add a connection icon to UsbCanAppViewUsbBusy (see usb_can_app_alloc()) via widget_add_icon_element
  2. add 2 string messages to UsbCanAppViewUsbBusy via widget_add_string_multiline_element
  3. call view_dispatcher_switch_to_view to display above elements

bool usb_can_scene_error_dialog_on_event ( void *  context,
SceneManagerEvent  event 
)

This function is called when a key is pressed by the user. If it is back key SCENE-START is re-entered.

This function is probably called by scene manager to evaluate if this scene shall be leaved (to return to SCENE-START): If event corresponds to a back key press:

  1. call scene_manager_previous_scene() to run previous scene (SCENE-START)
  2. if scene_manager_previous_scene() returned an error (i.e false) : stop app via scene_manager_stop and view_dispatcher_stop
    Returns
    true if key if back key was entered, false if another event has trigged this function.
