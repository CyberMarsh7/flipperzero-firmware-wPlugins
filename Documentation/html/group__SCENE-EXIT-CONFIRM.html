◆ usb_can_scene_exit_confirm_on_enter()

void usb_can_scene_exit_confirm_on_enter ( void *  context)

This function is called in SCENE-USB-CAN when user presses back key when in SCENE-USB-CAN.

This function will display exit confirm message by :

  1. registering app context to be passed to usb_can_scene_exit_confirm_dialog_callback() by calling dialog_ex_set_context()
  2. call dialog_ex_set_left_button_text() to create "Exit" button
  3. call dialog_ex_set_right_button_text() to create "Stay" button
  4. create exit message (""Exit USB-CAN?") by calling dialog_ex_set_header()
  5. register usb_can_scene_exit_confirm_dialog_callback() callback to be called when an oprion (button) is choosen through dialog_ex_set_result_callback()
  6. request gui component to previously described element by calling view_dispatcher_switch_to_view()

◆ usb_can_scene_exit_confirm_on_event()

bool usb_can_scene_exit_confirm_on_event ( void *  context,
SceneManagerEvent  event 
)

This function is trigged by usb_can_scene_exit_confirm_dialog_callback when an option is chosen ("Stay" or "Exit") to switch to next scene regarding option chosen ( SCENE-START or SCENE-USB-CAN).

This function will chose scene to go regarding event ( SceneManagerEvent) that trigged the function :

  • SCENE-START if "Exit" Button was chosen : SceneManagerEvent::event = DialogExResultRight ( SceneManagerEvent::type = SceneManagerEventTypeCustom)
  • SCENE-USB-CAN if "Stay" button was chosen : SceneManagerEvent::event = DialogExResultLeft
  • stay in SCENE-EXIT-CONFIRM if back key was pressed : SceneManagerEvent::type = SceneManagerEventTypeBack
    Returns
    True if back key was pressed (to prevent scene manager automatically restore SCENE-START), False otherwise
