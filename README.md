# Flipper-Zero-Blackjack

Blackjack! for your Flipper Zero

## Features
- Full Blackjack gameplay: 
  - Hit, Stand, and Bust!
  - Dealer follows traditional casino rules.
- User-friendly interface built for the Flipper Zero.
