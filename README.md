# Flipper Zero E220 LoraModule Configurator

E220-900T22S(JP), a configurator for LoRa modules used only in Japan.
I hope to support other modules in due course.

# How To Connect


```
FLI  | LORA
3.3V - Power
PA7 - M0
PA6 - M1
PA4 - AUX
TX - RX
RX - TX
GND -  GND
```
